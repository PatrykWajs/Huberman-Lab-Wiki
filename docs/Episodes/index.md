# Episode Library

394 Huberman Lab episodes — each with a structured summary and full transcript.

## How to Navigate

- **[Knowledge Map](../MAP.md)** — all episodes in chronological order with topic tags
- **[Guest Directory](../GUESTS.md)** — browse by guest expert
- **[Topic Conclusions](../Conclusions/index.md)** — aggregated findings by subject
- **[Mechanisms Index](../MECHANISMS.md)** — episodes organized by biological mechanism

## Episode Format

Each episode contains two files:

| File | Contents |
|------|----------|
| `summary.md` | Key findings, protocols, timestamps, and guest context |
| `transcript.md` | Full timestamped transcript with anchor links |

Summaries link directly to transcript segments for every finding.
